txt2struct.m
  struc = txt2struct(filename)
  Takes a file that looks like this:
   key1 = val1
   key2 = val2
   ...
  and returns a struct that looks like
   struc.key1 = 'val1'
   struc.key2 = 'val2'
   ...
  All keys and values are treated as strings.
